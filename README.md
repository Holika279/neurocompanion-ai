# 🧠 NeuroCompanion AI

An AI-based mental health companion designed to provide emotional support, track user mood, and offer intelligent responses.

## 🚀 Features
- 🤖 AI Chat Support
- 😊 Mood Tracking
- 📊 Dashboard for user insights
- 🔐 Secure Login & Authentication

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ⚙️ Installation
1. Clone the repository
2. Run npm install
3. Run npm start

## 📌 Use Case
Helps users manage stress and mental well-being using AI interaction.
